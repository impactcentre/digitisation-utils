# digitisation-utils
to your pom.xml and reference the projects as:

```xml
<dependency>
    <groupId>eu.digitisation.utils</groupId>
    <artifactId>input</artifactId>
    <version>0.2</version>
</dependency>
<dependency>
    <groupId>eu.digitisation.utils</groupId>
    <artifactId>log</artifactId>
    <version>0.2</version>
</dependency>
<dependency>
    <groupId>eu.digitisation.utils</groupId>
    <artifactId>text</artifactId>
    <version>0.2</version>
</dependency>
<dependency>
    <groupId>eu.digitisation.utils</groupId>
    <artifactId>xml</artifactId>
    <version>0.2</version>
</dependency>
```
